This program automatically searches for 30 numbers in Microsoft Edge to earn Microsoft Rewards points.

Files:
Script folder: Contains the Python Script in case you want to modify it.

Exe file: The ready-to-use .exe is in the main folder.

Usage and Safety:
Do not use the PC while the program is running to avoid input errors.

To stop the program, simply close the terminal window that opens at startup.

The program simulates keyboard inputs in order to do his job.
